# TaskbarLyrics 设置页视觉精修规范（精简版）

## 1. 当前阶段定位

当前项目已经具备：

- 深色主题
- Sidebar 导航
- 卡片系统
- Fluent 风格
- 毛玻璃背景
- WPF 基础架构

因此本次改造目标不是：

# 重做 UI

而是：

# 对已有 UI 进行视觉比例精修

重点：

- 组件比例
- 信息密度
- 控件厚度
- 视觉节奏
- 对齐与层级

---

# 2. 当前旧版 UI 的核心问题

## 2.1 Win11 设置页感过强

当前界面偏：

- 留白大
- 行高偏高
- 控件太薄
- 信息密度低
- 更像系统设置页

缺少：

- 专业工具感
- 开发工具气质
- 高密度结构感

---

# 3. 新版视觉目标

新版方向：

- 更像专业工具
- 更紧凑
- 更厚重
- 更结构化

参考风格：

- JetBrains
- OBS
- Steam Deck 设置
- 专业播放器工具
- 开发者工具

---

# 7. 视觉核心变化

本次改造真正的重点：

| 项目 | 旧版 | 新版 |
|---|---|---|
| 行高 | 松散 | 紧凑 |
| 控件 | 偏薄 | 更厚 |
| 信息密度 | 低 | 高 |
| 结构感 | 弱 | 强 |
| 工具感 | 一般 | 强 |
| 留白 | 偏多 | 克制 |
